Estimates an average required time to complete reading a post.

== Description ==

Estimating the reading time for your readers. The calculation based on standard reading speed.

== Installation ==

1. Download the plugin.

1. Extract all the files.

1. Upload everything to the `/wp-content/plugins/smart-reading-time`  directory.

1. Activate the plugin through the 'Plugins' screen in WordPress.

1. Use the Settings->Smart Reading Time to configure the plugin
